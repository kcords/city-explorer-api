# City Explorer

**Author**: Kyle Cordell
**Version**: 1.0.2

## Overview

This repo provides the back end API routes for the City Explorer application through Node.js

## Getting Started

Clone the repo, then run:

```
npm install
npm start
```

## Architecture

* Node.js
* Express.js
* Axios
